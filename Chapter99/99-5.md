### 整数线性组合

```
指的是用整数系数 x,y 表示 a,b 的线性组合：
```

$$
\begin{aligned}
&a\ *\ x\ +\ b\ *\ y\ =\ d\ =\ gcd(a,b)
\end{aligned}
$$

```
这个概念在求模逆元、解线性同余方程时至关重要

示例：写出 48 和 18 的最大公约数的整数线性组合

Step 1：先用欧几里得算法求 gcd(48,18) = 6

Step 2：回代求出整数 x,y
```

$$
\begin{aligned}
&6\ =\ 18\ -\ 1\ *\ 12\\
&=\ 18\ -\ 1\ \ *\ (48\ -\ 2\ *\ 18)\\
&=\ 18\ -\ 1\ \ *\ 48\ +\ 2\ *\ 18\\
&=\ 3\ *\ 18\ -\ 1\ *\ 48\\
\end{aligned}
$$

```
所以我们得到：
```

$$
\begin{aligned}
&6\ =\ (-1)\ *\ 48\ +\ 3\ *\ 18\\
\end{aligned}
$$

```
即 48x + 18y = 6，其中 x = -1，y = 3
```
