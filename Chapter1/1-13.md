打开文件看不到东西，Ctrl + A 全选却有东西

![](https://pic1.imgdb.cn/item/678501a5d0e0a243d4f3f1b2.jpg)

在 Whitespace 编码中，只有三种类型的字符被用来表示数据：

```
空格 ( ): 用来表示二进制中的 0
制表符 (\t): 用来表示二进制中的 1
换行符 (\n): 用来表示分隔符，即用来表示一个数据单元的结束
```

**Whitespace 编码特点：**

```
正常打开没有内容显示，全选却有东西
```
